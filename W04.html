This assignment introduces simplified scripting with jQuery.

The code is available at https://bitbucket.org/professorcase/W04.

Results of the unit tests are available at test/test.html.


Currently, the application gets the user's name and the size of their ranch. The app calculates square miles and estimates how many sheep they can hold on their land.


Your goal is to create a new app using jQuery and native JavaScript that calculates the number of seats reserved at the next Bearcats sporting event.


Your new application should:

  1. Request the name of the organization attending the event.
  2. Ask the type of sporting event (e.g. football, soccer, volleyball).
  3. Ask the number of rows the organization wants to reserve.
  4. Ask the number of seats per row the organization wants to reserve.
  5. Calculate the number of seats requested.
  6. Display the number of students that will have a reserved seat.
  7. Display one Bearcat logo for each reserved seat.
  8. Change the application name displayed in the menu to a suitable name for your application.

Click the button to start the app.
